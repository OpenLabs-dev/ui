---
title: Progress
description: A progress bar is a graphical UI representation to visualize the progression of an extended computer operation, such as a download, file transfer, or installation.
---

## Import

- **Progress**: The main component to display an progress.

## Usage

<Progress value={66} className="w-[60%]" />


```tsx
"use client"

import * as React from "react"
import { Progress } from "@openlite/ui"

export function ProgressDemo() {
  const [progress, setProgress] = React.useState(13)

  React.useEffect(() => {
    const timer = setTimeout(() => setProgress(66), 500)
    return () => clearTimeout(timer)
  }, [])

  return <Progress value={progress} className="w-[60%]" />
}
```

## API

### Progress Props

| Props | Type | Description |
| --- | --- | --- |
| `variant` | `'default' \| 'primary' \| 'success' \| 'warn' \| 'error'` | The variant of the progress. |
| `size` | `'sm' \| 'md' \| 'lg'` | The size of the progress. |
