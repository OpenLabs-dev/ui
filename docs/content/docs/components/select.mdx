## Import

- **Select**: The main component to display an select.
- **SelectContent**: The component that contains a group of select items.
- **SelectItem**: The component that represents a select item.
- **SelectTrigger**: The component active event Select.
- **SelectValue**: The component contains a value of select items.
- **SelectGroup**: The component can group a group of Select items.
- **SelectLabel**: The component can add a title to each group of Select items.
- **SelectSeparator**: The component can separate the Select groups.

## Usage

<Select>
    <SelectTrigger className="w-max">
        <SelectValue placeholder="Select a value" />
    </SelectTrigger>
    <SelectContent>
        <SelectItem value="option1">Option 1</SelectItem>
        <SelectItem value="option2">Option 2</SelectItem>
    </SelectContent>
</Select>

```tsx
import * as React from "react"

import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "@openlite/ui"

export function SelectDemo() {
  return (
    <Select>
        <SelectTrigger className="w-max">
            <SelectValue placeholder="Select a value" />
        </SelectTrigger>
        <SelectContent>
            <SelectItem value="option1">Option 1</SelectItem>
            <SelectItem value="option2">Option 2</SelectItem>
        </SelectContent>
    </Select>
  )
}
```
