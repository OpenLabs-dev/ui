## What is OpenUI?

Open UI is a design system built with Tailwind CSS and Radix UI. It provides a set of components and utilities that you can use to build your applications.

## Why OpenUI?

Open UI is built on top of Tailwind CSS, which means you can use all the utilities and components provided by Tailwind CSS. In addition, Open UI provides a set of components that are built with Tailwind CSS and Radix UI.

## Features

- **Components**: A set of components built with Tailwind CSS and Radix UI.
- **Utilities**: A set of utilities built with Tailwind CSS.
- **Dark Mode**: A dark mode that you can enable with a single line of code.
- **Customization**: You can customize the design system to fit your needs.
